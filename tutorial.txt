Passos :
1. Dê npm init e responda as informações pendentes ( se não souber inglês, usa um tradutor )
2. Instale todas as depêndencias ( usando npm i nome da depêndencia )
3. Crie o seu bot no https://discordapp.com/developers/applications/
4. Crie seus comandos, eu deixei um exemplo lá na pasta comandos
5. Troque o Prefixo e o Token do seu bot na bot.js ( não sabe prefixos bons ? veja no final desse arquivo, lá dou exemplos )
6. Roda o bot com node . no Node.js command Prompt ( você precisa ter o Node.js instalado claro. )
